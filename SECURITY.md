# Security Policy

## Supported Versions

We currently support the latest `main` branch and the most recent tagged release for security updates. 

| Version | Supported          |
| ------- | ------------------ |
| v1.0.0+ | :white_check_mark: |
| < 1.0.0 | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within Sentinel, please do **NOT** open a public issue. 

Instead, please send an email to **maintainers@sentinel-mcp.com**. We will treat your report with the highest priority. 

When reporting, please include:
- A detailed description of the vulnerability.
- Steps to reproduce the issue.
- The version of Sentinel where this was observed.
- Any potential impact on user data or machine security.

You can expect an initial response within 48 hours. If the vulnerability is confirmed, we will provide a timeline for the fix and ensure you are credited in the release notes.
