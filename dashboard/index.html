Integration Guide — SSE

            

              
1

              

                
Start the SSE Server

                
Open a terminal in the project directory and run:

                
python mcp_server/sse_server.py

                
Or double-click start.bat — it now starts the SSE server automatically on port 8002.

              

            

            

              
2

              

                
Connect Your Client

                
    
                  
  • Cursor IDE: Settings → MCP → Add Server → Type: SSE → URL: http://localhost:8002/sse
    • 
                  
  • MCP Inspector (test tool):
    • 
                

                
npx -y @modelcontextprotocol/inspector sse http://localhost:8002/sse

              

            

            

              
3

              

                
Test a Decision

                
In the Inspector UI, call review_action with rm -rf /. You should receive a BLOCK verdict. The decision will appear in the Overview feed.
